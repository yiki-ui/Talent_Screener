## GEMINI SERVICE — EXACT PROMPT TO USE

Build geminiService.ts with this prompt template:

```
You are an expert AI recruiter evaluating candidates for a job opening.
Evaluate ALL candidates below against the job requirements and return a ranked shortlist.

=== JOB DETAILS ===
Title: {jobTitle}
Description: {jobDescription}
Required Skills: {requiredSkills}
Nice to Have: {niceToHaveSkills}
Experience Required: {minYearsExperience}+ years ({experienceLevel} level)
Education Requirement: {educationRequirement}

=== CANDIDATES ===
{candidatesJSON}

=== SCORING WEIGHTS ===
- Skills match: 40% (how many required skills does the candidate have?)
- Experience: 30% (years of experience vs requirement)
- Education: 15% (relevance of degree/institution)
- Overall fit: 15% (bio, current role, career trajectory)

=== INSTRUCTIONS ===
1. Score every candidate on all 4 dimensions (0-100 each)
2. Calculate weighted total matchScore
3. Rank all candidates from highest to lowest matchScore
4. For the top {shortlistSize} candidates, provide detailed reasoning
5. For candidates outside the shortlist, still include score but reasoning can be brief

Return ONLY valid JSON — no markdown, no explanation, no preamble:
[
  {
    "rank": 1,
    "candidateId": "string",
    "name": "string",
    "matchScore": 87,
    "skillsScore": 90,
    "experienceScore": 85,
    "educationScore": 80,
    "fitScore": 88,
    "strengths": [
      "Has 5 of 6 required skills including React and TypeScript",
      "4 years experience exceeds the 3-year minimum",
      "Previous role at a fintech startup is highly relevant"
    ],
    "gaps": [
      "No GraphQL experience listed (nice-to-have)",
      "No mention of team leadership despite senior level req"
    ],
    "recommendation": "Strong candidate. Technical skills are well-aligned. Recommend for first-round interview.",
    "shortlisted": true
  }
]
```

## API ROUTES

### Jobs
POST   /api/jobs              — create job
GET    /api/jobs              — list all jobs
GET    /api/jobs/:id          — get job detail
PUT    /api/jobs/:id          — update job
DELETE /api/jobs/:id          — delete job

### Applicants
POST   /api/applicants/json          — ingest JSON array of Umurava profiles
POST   /api/applicants/csv/:jobId    — upload CSV file
POST   /api/applicants/pdf/:jobId    — upload single PDF resume
GET    /api/applicants/:jobId        — list applicants for a job
DELETE /api/applicants/:id           — remove applicant

### Screening
POST   /api/screening/run           — trigger AI screening { jobId, shortlistSize }
GET    /api/screening/:screeningId  — get screening results
GET    /api/screening/job/:jobId    — get all screenings for a job
GET    /api/screening/:screeningId/export  — export shortlist as CSV
